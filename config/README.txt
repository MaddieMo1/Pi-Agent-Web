这里用于放置启动时可选读取的本地配置文件。

Tavily API Key 配置方法：

1. 复制 tavily-api-key.example.txt，并重命名为 tavily-api-key.txt。
2. 在 tavily-api-key.txt 的第一行填写真实 Tavily API Key。
3. 重新运行“启动 Pi Agent 国内模式.bat”或“启动 Pi Agent.bat”。

注意：

- tavily-api-key.example.txt 是示例文件，可以放进公开压缩包。
- tavily-api-key.txt 是本地私有密钥文件，不要提交到公开仓库，也不要发给其他人，除非你明确希望别人使用你的 key。
