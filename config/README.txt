Optional local configuration files for startup.

Tavily API key:

1. Copy tavily-api-key.example.txt to tavily-api-key.txt.
2. Put the Tavily API key on the first line.
3. Restart with "启动 Pi Agent 国内模式.bat" or "启动 Pi Agent.bat".

Do not share tavily-api-key.txt in a public zip unless you intentionally want others to use that key.
